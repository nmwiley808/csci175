## Course Description

CSCI 175 focuses on **computational methods used in bioinformatics** and the application of algorithms to biological data.

Topics covered in this course include:

- Biological sequence alignment algorithms  
- Sequence alignment heuristics  
- Bioinformatics software tools  
- Combinatorial pattern matching  
- Motif discovery and motif finding algorithms  
- Gene prediction methods  
- Gene expression measurement techniques  
- High-throughput sequencing technologies  
- Data analysis methods used in molecular biology and microbiology  
- Selected topics from current bioinformatics research  

The course combines **algorithm design, computational biology, and data analysis techniques** used in modern biological research.
